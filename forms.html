Forms

Forms are a main part of all the Greenlight Products and it is important that elements are consistant between each of the products. Forms should be in a single column where possible, or 2 columns where space is very tight.

 

Input Boxes

The input title appears first.
Inputs should appear directly beneath the input title.
A short sentence indicating what a user needs to input should appear underneath the input box.

Asterix

Mandatory fields are indicated with a red asterix (#cc00)
